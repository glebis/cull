# Undo Interface Prototypes

Static SVG design prototypes for Cull's Tokyo Night desktop UI. These are visual-only explorations for undo and redo workflows; they do not define implementation details.

- `01-status-chip-toast.svg` explores compact feedback after undo and redo actions.
- `02-command-palette-history-preview.svg` explores reversible history inside the command palette with a preview pane.
- `03-undo-history-popover.svg` explores a dedicated toolbar popover for undo history.

All three prototypes are standalone `1600x1000` SVG files intended to open directly in Cull.
`03-undo-history-popover.svg` includes explicit vector/text rendering hints and layered SVG
effects for a higher-quality direct render without embedding raster artwork.
