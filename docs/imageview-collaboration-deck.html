Why Not Just Use Miro?

  • $8-16/user/month kills community and workshop budgets
  • 10% feature usage — overwhelming UI for non-tech participants
  • Not built for images — great for sticky notes, bad for photo curation
  • No self-hosting — dealbreaker for GDPR-conscious orgs
  • No offline/local mode — breaks in gallery Wi-Fi dead zones
"Most teams use 10% of the features and pay for 100%."

How a Session Works

Host

  1. 01 Click "Start Session"
  2. 02 QR code + 3 links generated
  3. 03 Select images, start review
  4. 04 See consensus emerge live
  5. 05 Apply selections to DB

Participants

  1. 01 Scan QR or open link
  2. 02 Enter name — no account needed
  3. 03 See images, rate independently
  4. 04 Comment, flag, compare
  5. 05 Audience: bookmark, buy
